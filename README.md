# FavoriteShowsApplication
Android Application to Browse TV Show from TV Maze  
   
**Demo:**  
[![Favorite Show Application](https://user-images.githubusercontent.com/90371913/152652887-7fcfcd75-2ec2-4201-9ef3-0f26604d2864.jpg)](https://youtu.be/J0dC9BAq1b4)  


Using **Kotlin**  

This Code Provide Simple **Android Application** For **Browsing and Saving Favorite TV Shows**

This Application Called **Favorite Shows Application**

This Application Uses **https://www.tvmaze.com/api** API  
This Application Uses **SQLite**
&nbsp;  
&nbsp;  

**This Application Will:**
* • Search TV Show from TV Maze API
* •	Get TV Show from TV Maze API
* •	Browse TV Shows as List
* • Save Favorite TV Shows in SQLite Database
* • Browse Favorite TV Shows from SQLite Database
* • Display Favorite TV Shows Details
* • Delete TV Shows from SQLite Database
